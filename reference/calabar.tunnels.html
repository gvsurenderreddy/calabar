Tunnels - calabar.tunnels¶

calabar.tunnels

This module encapsulates various tunnel processes and their management.

exception calabar.tunnels.ExecutableNotFound¶
The given tunnel executable wasn’t found or isn’t executable.
exception calabar.tunnels.TunnelTypeDoesNotMatch¶
The given tun_type doesn’t match expected Tunnel.
exception calabar.tunnels.TunnelsAlreadyLoadedException¶
Once tunnels are loaded the first time, other methods must be used to update them
calabar.tunnels.get_tunnels(config)¶

Return a dictionary of dictionaries containg tunnel configurations based on the given SafeConfigParser instance.

An example return value might be:

{
    'foo':
        {
            'tunnel_type': 'vpnc',
            'conf_file': '/etc/calabar/foo.conf',
            'ips': [10.10.254.1]
        },
    'bar':
        {
            'tunnel_type': 'ssh',
            'from': 'root@10.10.251.2:386',
            'to': '127.0.0.1:387
        }
}
calabar.tunnels.parse_tunnel(config, section)¶
Parse the given section in the given config ConfigParser.ConfigParser object to generate a tunnel configuration dictionary using all configured tunnel types and their configuration parsers.
